### 介绍

godot是一个功能齐全的,开源的,麻省理工学院许可,游戏引擎。它关注有很好的工具,面向视觉的工作流可以导出到电脑,手机和网络平台,没有麻烦。编辑、语言和api功能丰富,但简单的学习,让你成为高效的在几小时内。

### 关于

Godot 几年前由 Juan Linietsky 和 Ariel Manzur 开发出来.
被OKAM Studio(http://www.okamstudio.com)发展.
