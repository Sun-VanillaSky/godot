### 介绍

godot是一个功能齐全的,开源的,麻省理工学院许可,游戏引擎.

### 关于

Godot几年前由Juan Linietsky和Ariel Manzur开发出来.
被OKAM Studio(http://www.okamstudio.com)发展.
